# gastest

## wordfreq — Word Frequency Counter

A simple Python CLI tool that reads a text file and prints the most common words with their counts.

### Usage

```bash
# Print top 10 words (default)
python3 wordfreq.py sample.txt

# Print top N words
python3 wordfreq.py sample.txt --top 5
```

### Example output

```
     8  the
     5  fox
     5  dog
     4  a
     3  quick
     3  lazy
     3  and
     3  brown
     2  stories
     2  time
```

### Requirements

- Python 3.6+
- No external dependencies
